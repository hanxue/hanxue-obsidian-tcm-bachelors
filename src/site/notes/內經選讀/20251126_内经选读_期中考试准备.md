# 阴阳应象大论
- ==阴阳者，天地之道也，万物之纲纪，变化之父母，生杀之本始，神明之府也，治病必求于本。==
	- 阴阳是归纳事物的纲领
	- 变化之父母：事物变化的根源
	- 生杀之本始：事物产生与消亡的缘由
	- 神明之府：万物运动变化内在场所
	- 本：阴阳

* 阴味出下窍，阳气出上窍。味厚者为阴，薄为阴之阳，气厚者为阳，薄为阳之阴。
